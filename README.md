Описание проекта
Многостраничное портфолио разработчика, демонстрирующее навыки через практическую реализацию:
1. Адаптивный дизайн с переключением темы
2. Пять разделов с разным функционалом
3. Интерактивная галерея фотографий
4. Три мини-игры с сохранением рекордов
5. Чистая CSS-анимации
